# DemoQA-Automation-WWG
Automated end-to-end tests for DemoQA using Behave BDD, Selenium WebDriver, and Page Object Model (POM) for maintainable test automation.

Features
Page Object Model: Clean separation of locators and page actions

Behave BDD: Readable Gherkin feature files with step definitions

Explicit Waits: Reliable element interactions without time.sleep()

Modular Design: Easy to add new pages and test scenarios
